name: XML Document Transformation with Conditional Random Fieldsurl: http://www.springerlink.com/content/r8048wu586036746public_in: Comparative Evaluation of Xml Information Retrieval Systems, International Workshop of the Initiative for the Evaluation of Xml Retrieval, Inex, Dagstuhl Castle, Germany, December, Revised &amp; Selected Papersauthors: Rémi Gilleron,Florent Jousse,Isabelle Tellier,abstract: We address the problem of structure mapping that arises in xml data exchange or xml document transformation. Our approach relies on xml annotation with semantic labels that describe local tree editions. We propose xml Conditional Random Fields (xcrfs), a framework for building conditional models for labeling xml documents.
