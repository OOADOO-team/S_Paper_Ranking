### 对远程仓库的更新
两种方式
* （建议）新的pull request，团队讨论确认后merge。
* （也可以）直接push。有可能会覆盖掉别人修改的东西。所以使用这种方式的时候要确认你本地代码和仓库中最新的一致。也就是在push之前备份好你做的updates然后pull再把updates加进来然后push
